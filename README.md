# Football Connection Game

This is a site which focuses on footballers and their connecting team mates. Users may try to guess connections between two specified or random players, or use the connector tool to find the shortest path between player A and player B.

The database stores all players who have played in the following leagues from 1990-2025:

- Premier League
- Championship
- Bundesliga
- Bundesliga 2
- La Liga
- La Liga 2
- Ligue 1
- Ligue 2
- Serie A
- Serie B
- Eredivise
- Liga Portugal
- Scottish Premiership
- Jupiler Pro League
- Super Lig


## Built With

- **React** - Frontend framework
- **Tailwind CSS** - Styling
- **Node.js / Express** - Backend
- **Python / BeautifulSoup** - Data Collection / Web Scraping
- **Docker** - Containerization
- **Neo4j** - Database
